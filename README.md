# Proyecto-Python
Proyecto Final de la materia de Programación

Proyecto donde se trabajaron scripts que realizan web scraping de información acerca de un deportista, en este caso el canelo.

El código funciona como una aplicación o base de datos que busca su info más relevante, 
información básica, estadísticas de peleas, noticias recientes y su fecha de su siguiente pelea.

El código trabaja web scraping, edición de archivos de texto, creación de archivos en excel y estructuras de datos en python.
